## How to use this directory

Open Claude Code inside this directory. Claude reads the master `claude-exam-prep/CLAUDE.md` and this exam's `CLAUDE.md`. It will help you:

1. Audit the source material in `raw/`.
2. Build the module tracker.
3. Teach every module in natural prose with rendered diagrams.
4. Generate a PDF per module and a cheat sheet per professor.
